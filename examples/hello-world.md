# Hello World CLI

A minimal example PRD to test Ralph. Creates a simple CLI tool.

## Tasks

### T1: Create main.rs
Create `src/main.rs` with a basic "Hello, world!" program that accepts an optional `--name` flag.

### T2: Add argument parsing
Use the `clap` crate to parse `--name <NAME>` from the command line. Default to "world" if not provided.

### T3: Add tests
Add a test that verifies the greeting format is correct.
